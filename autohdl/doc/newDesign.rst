Создание нового проекта
===================================

Новый проект создается с помощью утилиты hdl.py.

.. note::

  python должен быть добавлен в переменную среды PATH.

Создание структуры проекта в текущей директории:

>>> hdl.py

Создание папки и в ней структуру проекта:

>>> hdl.py -n <project_name>

Также при генерации проекта инициализируется гитовый репозиторий
